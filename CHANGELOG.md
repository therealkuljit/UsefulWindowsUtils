# Changelog

## 1.0.1

- Shortened app branding to `UsefulWindowsUtils` across the app and docs.
- Added rounded theme-aware controls for buttons, tick boxes, radio buttons, toggles, progress bars, scrollbars, search fields, and logs.
- Added multi-select uninstall support in Installed Apps, including selecting apps by clicking their rows.
- Updated Installed tab section controls so Uninstaller and Debloater match the refreshed toolbar button style.
- Improved C2 Collector layout so the log panel remains visible beneath compact top controls.
- Improved live language switching for navigation and primary app actions.
- Removed obsolete runtime image-control assets and the old bundled font.
