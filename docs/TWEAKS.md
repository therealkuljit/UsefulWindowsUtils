# Windows Tweaks Catalog

Tweaks are loaded from `config/tweaks.json`. High-risk items are highlighted in red in the app. Review registry/service/script actions before applying.

## Presets

- Minimal
- Standard

## Essential Tweaks

- Activity History - Disable: Erases recent docs, clipboard, and run history.
- Hibernation - Disable: Turns off hibernation and related Explorer power-menu visibility.
- Widgets - Remove: Removes Widgets platform/runtime packages.
- Start Menu Previous Layout - Enable: Restores an older Start Menu layout where supported.
- Microsoft Store Recommended Search Results - Disable: Blocks Store recommendations in Start search.
- Location Tracking - Disable: Disables Windows location tracking settings/services.
- Services - Set to Manual: Reduces selected service startup behavior.
- ConsumerFeatures - Disable: Blocks automatic consumer apps and suggestions.
- Telemetry - Disable: Disables Microsoft telemetry settings.
- Delivery Optimization - Disable: Stops peer upload/download sharing behavior.
- BitLocker - Disable: Disables BitLocker related behavior.
- Unwanted Pre-Installed Apps - Remove: Removes bundled Windows apps.
- Restore Point - Create: Creates a restore point before changes.
- End Task With Right Click - Enable: Enables taskbar end-task option.
- Windows Platform Binary Table (WPBT) - Disable: Blocks vendor WPBT execution behavior.
- Disk Cleanup - Run: Runs Disk Cleanup for drive C.
- Temporary Files - Remove: Clears temp folders.
- File Explorer Automatic Folder Discovery - Disable: Disables automatic folder type discovery.
