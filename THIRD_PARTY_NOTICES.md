# Third-Party Notices

This project can interact with third-party tools, package managers, APIs, and assets.

## Package Managers and Windows Tools

The app may invoke:

- Winget
- Chocolatey
- PowerShell
- DISM
- Defender PowerShell cmdlets
- Windows Registry APIs
- Windows service tools
- Oscdimg from the Windows ADK, when installed

These tools are governed by their respective licenses and terms.

## Threat-Intelligence APIs

The app can query configured provider APIs:

- VirusTotal
- ThreatFox
- URLhaus
- MalwareBazaar
- AlienVault OTX
- Pulsedive
- Hybrid Analysis

Use of those services is subject to their own terms, privacy policies, quotas, and acceptable-use policies.
